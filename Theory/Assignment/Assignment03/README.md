# Student Firebase Database App

## Project specification
1. NSU ID
2. Student Name
3. School Name
4. Department Name
5. Phone Number
6. NID Number
7. Present Address
8. Permanent Address
9. Room Database
10. Recycler view list
11. Pop up messages
